Color Palette Guide

A clear, friendly, and high-contrast palette for AI agents built for full-site use across panels, buttons, cards, banners, and status surfaces.

Primary Palette

Use one dominant color per state and keep the rest muted.

Usage Rules

Keep the interface predictable.

  • • Apply the palette to surfaces, badges, banners, buttons, and icons—not only text.
  • • Use soft tints for large backgrounds and stronger shades for primary actions or status panels.
  • • Keep one dominant accent per section and use neutrals for the rest of the interface.

Accessibility

Readable, friendly, and inclusive.

  • • Keep text on colored surfaces at a contrast ratio above WCAG AA.
  • • For a truer yellow look, use amber-400 or amber-500 as an accent and pair it with dark text.
  • • Pair icons with labels where possible and never rely on color alone.
